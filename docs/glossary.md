# Glossary

- **Execution Fingerprint**: Deterministic hash linking execution intent and runtime context.
- **Attestation Hash**: Integrity digest for wrapper identity and execution evidence metadata.
- **Canonical Telemetry**: Normalized, schema-bound event representation used for signing.
- **Federation Core**: Cross-organization governance layer for policy and evidence exchange.
- **Evidence Store**: Tamper-evident storage layer for correlated assurance records.
